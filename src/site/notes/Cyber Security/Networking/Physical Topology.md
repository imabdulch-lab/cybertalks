&nbsp;
Physical topology defines the physical layout of a network. It shows how various devices are connected to each other in a network, either through cables or wireless connections.
&nbsp;
# **1. Point-to-Point Topology (P-to-P / P2P)** 

&nbsp;
Two nodes are linked directly.
![](https://i.postimg.cc/pTqqdVbq/P2P-Topology.jpg)
&nbsp;
# **2. Bus Topology**

&nbsp;
Devices are connected along a single cable.
![](https://i.postimg.cc/sXww2fd0/Bus-Topology.jpg)
&nbsp;
# **3. Star Topology** 

&nbsp;
All devices connect to a central hub or switch.
![](https://i.postimg.cc/mD88rZfm/Star-Topology.jpg)
&nbsp;
# **4.Tree  Topology (Hierarchical)**

&nbsp;
The combination of bus and star topology.
![](https://i.postimg.cc/C1mmKMTv/Tree-Topology.jpg)
&nbsp;
# **5. Ring Topology** 

&nbsp;
Devices form a circular connection where data travels in one direction.
![](https://i.postimg.cc/9MJJfX5n/Ring-Topology.jpg)
&nbsp;
## *i. Single Ring (Uni-directional)*
Data flows in only one direction—either clockwise or counter-clockwise.
&nbsp;
## *ii. Dual Ring (Bi-directional)*
Two connections between nodes allow data to flow in both directions.
&nbsp;
 ![](https://i.postimg.cc/26Ymdwt5/Uni-Bi-Ring.jpg)
 &nbsp;
# **6. Mesh Topology** 

&nbsp;
Each node connects to every other node. 
![](https://i.postimg.cc/Y9bbS2BK/Mesh-Topology.jpg)
&nbsp;
## *i. Partial Mesh*
All the nodes aren't necessary to be connected with one another.
&nbsp;
## *ii. Full Mesh* 
All the nodes within the network are connected with every other.
&nbsp;

![](https://i.postimg.cc/sfpGRHyr/Partial-Full-Mesh.jpg)
&nbsp;
# **7. Hybrid Topology** 
&nbsp;
A combination of two or more topologies.
![](https://i.postimg.cc/wv00BxYY/Hybrid-Topology.jpg)
